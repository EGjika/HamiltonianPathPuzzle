How to play:

Tap (or click) cells to trace your own path.
Path must move 1 step up / down / left / right.
Fill every cell exactly once and follow the numbered order.
Works with touch screens: just tap on each next cell.
This puzzle is based on Hamiltonian path concepts.
This version is original and independently built in R Shiny.

https://shareknowledge2learn.shinyapps.io/Hamiltonian_Shiny/
